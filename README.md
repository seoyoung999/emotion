# 😊 Emotion

## 📌 프로젝트 소개
**Emotion**은 텍스트 데이터를 기반으로  
**감정 분류(Emotion Classification)**를 수행하는 프로젝트입니다.

입력된 문장의 감정을 분석하여  
기쁨, 슬픔, 분노 등과 같은 감정 상태를 분류하는 것을 목표로 하며,  
자연어 처리(NLP)와 머신러닝 흐름을 학습하기 위해 제작되었습니다.

---

## 🛠 사용 기술
- **Language**: Python
- **Library / Framework**
  - scikit-learn
  - pandas
  - numpy
- **IDE**: VS Code
- **Version Control**: Git, GitHub

---

## ✨ 주요 기능
- 텍스트 데이터 전처리
- 감정 분류를 위한 머신러닝 모델 학습
- 학습된 모델을 활용한 감정 예측
- 데이터 기반 감정 분석 결과 도출

- 감정 분석(Emotion Analysis)의 기본 개념
- Python 기반 데이터 처리 경험
- GitHub를 활용한 프로젝트 버전 관리 경험
